Thinpad RISC-V CPU
---------------

2020年秋季学期，计算机组成原理大作业，基于RISC-V指令集的多周期CPU。

[支持](./doc/plan.md)功能：

- 19 条基本指令和 3 条附加指令
- 异常、中断
- MMU
- ....
